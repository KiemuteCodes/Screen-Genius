About Us

When you feel like you’ve already seen everything good, Screen Genius will find you something to watch. The Screen Genius website will find titles you haven’t seen before when the algorithmic methods the big streaming services use fail. When you’ve found the just what you want to watch, click the link to take you to the streaming service that offers it.

We are currently working on adding new features to Screen Genius! These feature include better suggested titles based on user input. A seperate e-petition page that will allow the user to request a streaming service to add a movie or show to their library.
